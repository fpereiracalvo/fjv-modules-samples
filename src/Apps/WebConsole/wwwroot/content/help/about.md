# About Fjv Web Console
## Web console sample for Fjv.Modules library
This application is a web command line console that use **Fjv.Modules** and **Fjv.Modules.DependencyInjection** to execute arguments passed to the prompt.

[![NuGet](https://img.shields.io/nuget/v/Fjv.Modules.svg)](https://www.nuget.org/packages/Fjv.Modules/) [![NuGet](https://img.shields.io/nuget/dt/Fjv.Modules.svg)](https://www.nuget.org/packages/Fjv.Modules/) [![License](https://img.shields.io/github/license/fpereiracalvo/fjv-modules.svg)](https://github.com/fpereiracalvo/fjv-modules/blob/main/LICENSE)

## Libraries

- SignalR 7.0.10 for real time communication.
- Marked latest for markdown rendering.
- Bootstrap v5.0.2 for styles.
- jQuery v3.6.0 for DOM manipulation.
